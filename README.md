# flul-cpp-dojo

A C++ experimentation platform for exploring language features, design patterns, and programming katas.

## Quick Start

```bash
cmake --preset debug
cmake --build --preset debug
./build/debug/core/runner/runner
```

## Structure

- `core/` - Dispatcher and runner infrastructure
- `dojo/` - Experiments organized by category (features, patterns, katas, sandbox)
